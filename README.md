# CareerBot AI

An AI-powered career assistant that helps users with career guidance, role suggestions, and decision-making support using LLMs.

##  Live Demo

👉 https://huggingface.co/spaces/navya-sharma16/CareerBot

##  Features

* Interactive AI chatbot for career advice
* Personalized suggestions using OpenAI APIs
* Simple UI (Gradio)
* Real-time responses

##  Tech Stack

* Python
* Gradio
* OpenAI API
* Hugging Face Spaces (Deployment)

## Run Locally

```bash
git clone https://github.com/navya16sharma/career-bot.git
cd career-bot
pip install -r requirements.txt
python app.py
```
